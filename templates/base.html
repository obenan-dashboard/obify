Obify Documentation

Getting Started

Obify is a powerful tool for comparing the performance of various AI models on customer review analysis tasks. Follow these steps to get started:

  1. Upload Reviews: Start by uploading a CSV or TXT file containing customer reviews.
  2. Configure Test: Select which AI models you want to test and customize the prompt template.
  3. Run Analysis: Submit your configuration to start the analysis process.
  4. View Report: Once complete, a detailed report will show you which models performed best based on various metrics including accuracy, cost, and speed.

Supported File Formats

  • CSV Files: Your CSV should contain a column with reviews. Obify will attempt to automatically detect the review column.
  • TXT Files: Plain text files with one review per line or reviews separated by blank lines.

Supported AI Models

Obify supports a wide range of AI models from different providers:

  • OpenAI: GPT-4, GPT-4o, GPT-4.1, GPT-4.1-mini, GPT-4o-mini, GPT-3.5-turbo
  • Anthropic: Claude 3 Sonnet, Claude 3.5 Sonnet, Claude 3 Opus, Claude 3 Haiku, Claude 3.5 Haiku
  • Google: Gemini Pro, Gemini 2.5 Pro
  • DeepSeek: DeepSeek v3

API Keys

To use the various AI models, you need to provide API keys in a config.json file or as environment variables:

  • OPENAI_API_KEY: For OpenAI models
  • ANTHROPIC_API_KEY: For Anthropic Claude models
  • GOOGLE_API_KEY: For Google Gemini models
  • DEEPSEEK_API_KEY: For DeepSeek models

Prompt Templates

The prompt template defines how Obify will instruct AI models to analyze reviews. Use the {review} placeholder to indicate where the review text should be inserted.
